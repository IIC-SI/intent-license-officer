# ILO Architecture Overview

## Components

- Custom GPT (Intent License Officer)
- Public Knowledge Base
- Intent Validation Test (PDF)
- Private Evaluation Engine
- Certificate Generator
- Public Registry
- Community Review Board (CRB)

## Data Flow

Applicant → Intake → 45Q Test → Evaluation → Decision → Certificate / Failure ID → Registry

## Security Boundaries

- Public repo: documentation only
- Private systems: evaluation, logs, keys
