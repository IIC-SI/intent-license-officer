# Contributing to Intent License Officer (ILO)

Thank you for your interest.

## What You Can Contribute

- Documentation improvements
- Ethical case studies
- Governance tooling
- Automation scripts
- Registry transparency tools

## What You CANNOT Contribute

- Changes to hidden scoring logic
- Attempts to weaken safeguards
- Content designed to game the test

## Workflow

1. Fork the repository
2. Create a feature branch
3. Make changes with clear rationale
4. Open a Pull Request using the template

## Review Requirements

- Documentation changes → 1 maintainer approval
- Governance changes → CRB review required
- Security-relevant changes → private review

## Style Guidelines

- Markdown only (`.md`)
- Clear headings
- No marketing language
- Ethical tone over promotional tone
