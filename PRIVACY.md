# Privacy Policy

## Data We Handle

Public repository:
- Documentation only
- No personal data

Operational systems (not in this repo):
- Applicant responses
- Evaluation metadata
- Registry records

## Retention

- Evaluation logs: default 7 years
- Public registry: metadata only
- Applicant data: private and access-controlled

## Consent

Applicants must explicitly consent before testing.

## Deletion Requests

Handled on a case-by-case basis, subject to audit obligations.
