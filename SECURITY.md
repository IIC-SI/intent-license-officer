# Security Policy

## Reporting a Vulnerability

⚠️ Do NOT open a public issue.

To report a vulnerability:
1. Email the project security contact
2. Include a detailed description
3. Provide reproduction steps if applicable
4. Describe potential impact

You will receive acknowledgment within 72 hours.

## Scope

This includes:
- Certificate generation
- Registry integrity
- Private applicant data
- Governance tooling

## Disclosure

We follow responsible disclosure practices.
